## 구현 결과 설명

금일 가계부 프로젝트 구현 결과 코드를 GitHub에 업로드했다.

이번 구현에서는 현재 자산을 입력하고, 소비 내역을 추가하면 목록에 표시되도록 구성했다. 또한 각 소비 내역의 시간, 항목명, 카테고리, 금액, 남은 자산이 화면에 보이도록 구현했다.

추가로 소비 내역 삭제 기능도 적용하여, 삭제 버튼을 누르면 해당 내역이 제거되고 화면이 다시 렌더링되도록 만들었다. 데이터는 브라우저 저장소를 활용해 새로고침 후에도 유지될 수 있도록 처리했다.

이번 작업을 통해 `import/export`를 이용한 모듈 분리, 컴포넌트별 파일 관리, 배열 `map()`을 이용한 목록 렌더링, `sessionStorage`를 활용한 데이터 저장 방식을 연습할 수 있었다. 또한 실행 과정에서 경로 문제와 확장자 문제를 수정하면서 ES Module 방식의 import 규칙도 함께 확인했다

## 구현 결과

가계부 구현 결과 코드를 GitHub에 업로드하였다.

현재 자산 입력, 소비 내역 추가, 소비 내역 목록 출력, 소비 내역 삭제 기능을 구현하였으며, 금액과 남은 자산이 화면에 정상적으로 반영되도록 작성하였다. 또한 브라우저 저장소를 사용하여 데이터가 유지되도록 처리하였다.

이번 구현을 통해 모듈 분리 방식과 이벤트 처리, 배열 데이터 렌더링, 저장소 연동 방식을 실습할 수 있었다.
